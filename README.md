## Clutch Gitleaks VS code extension

### Installation

```
npm install -g vsce
npm install -g yo generator-code
npm install
vsce package
```

```
code --install-extension clutch-scan-0.0.1.vsix
code --uninstall-extension clutch-scan-0.0.1.vsix
```
